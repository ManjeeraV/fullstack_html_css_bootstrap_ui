This project focuses on the User Interface development of an online food ordering application that offers users a convenient
and visually appealing platform to browse, select, and place orders for food items. The application will
showcase a menu, allow users to input order details through a form, and display an order summary. This
static application focuses on design, layout, and responsive behavior using HTML, CSS, and Bootstrap.

The project focuses on ability to organize content, integrate user inputs, and implement responsive
design for a seamless user experience.


The UI application involves the website sections -

1. Navigation Bar –
Create a responsive navigation bar that includes:
● A brand name (For example: "Foodie's Delight").
● Navigation links for Home, Menu, and Order Now sections.
● A collapsible menu for smaller screens using Bootstrap's navbar utilities.

2. Hero Section -
The homepage should include:
● A hero banner with a title, tagline, and a call-to-action button linking to the Menu

3. Menu Section -
Showcase the menu items in a grid layout. Each food item should include:
● An image.
● A name and price.

4. Order Form Section -
Design an order form where users can place orders. The form should include:
● Name and Email input fields.
● A delivery address textarea.
● A dropdown menu to select food items.
● A button to Place Order (non-functional).

5. Order Summary Section -
Create a table to summarize the orders. The table should include:
● Food item name, quantity, price per item, and total price for each item.
● A header row with appropriate column names.

6. Footer -
● Add a footer at the bottom of the page with Copyright information.
